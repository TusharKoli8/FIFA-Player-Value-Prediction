# FIFA Player Value Prediction 

##  Overview
This project predicts the market value of football players using machine learning based on their physical, technical, and performance attributes.


##  Dataset
Includes player features like:
- Physical attributes (age, height, weight)
- Technical skills (dribbling, passing, ball control)
- Performance metrics (reactions, composure)
- Target: Player market value


##  Approach
- Data cleaning and preprocessing
- Handling missing values and encoding
- Feature scaling using StandardScaler
- PCA for dimensionality reduction
- Models used:
  - Linear Regression
  - Random Forest


##  Results
- Random Forest performed best
- PCA reduced dimensions but slightly lowered accuracy
- Key factors: reactions, composure, ball control


## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn


##  Conclusion
Machine learning models can effectively predict player value, with Random Forest giving the best performance.
